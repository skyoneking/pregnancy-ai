# 已知问题和限制

## 已知问题

### 1. 覆盖率工具解析错误
- `app/onboarding/page.tsx` 在 Rollup (coverage) 中有解析问题
- 影响：无法生成精确的覆盖率报告
- 状态：不影响功能，覆盖率通过测试比例估算约 85%

### 2. 4 个测试文件导入错误
- `onboarding.test.tsx`、`page.test.tsx`、`profile.test.ts`、`chat.route.test.ts` 因环境变量或模块删除导致导入失败
- 影响：这些是旧的测试文件，不影响新功能的 184 个测试
- 建议：清理这些过时的测试文件

### 3. 登录 API 响应时间较长
- 登录接口约 2.2s 响应（Supabase 云端认证）
- 原因：Supabase Auth 服务在海外，网络延迟
- 缓解：已有加载状态 UI 提示

## 功能限制

### 认证系统
- 不支持短信验证码（使用手机号+密码方案）
- 不支持第三方登录（微信、支付宝等）
- 手机号格式仅支持中国大陆 11 位号码（1 开头）
- 密码重置通过旧密码验证，不支持邮件/短信重置

### 知识库
- 知识内容为静态数据，存储在代码中而非数据库
- 孕期知识覆盖 1-42 周，但部分周数内容较简略
- 食物安全数据有限，未知食物返回"未知"等级
- 内容为中文，不支持多语言

### 多设备同步
- 依赖 Supabase Realtime 服务，离线时无法同步
- 推送频率设置存储在 localStorage，不跨设备同步

### AI 对话
- 使用智谱 GLM-4-Flash 模型，响应质量依赖模型能力
- 对话历史不持久化（页面刷新后丢失）
- 无法上传图片或文件
